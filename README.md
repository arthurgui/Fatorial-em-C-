# Fatorial-em-C-

A criação do projeto foi feita no Visual studio community 2019


Desafio:
>>>> Escreva uma função f que mostre a soma de todos os dígitos de um fatorial, 
que varia de 0 a 2000 (em termos matemáticos - [0,2000]).


Ao baixar o projeto, encontrar na primeira pasta com o nome BRF1, a qual está localizada o
trecho de codigo feito em C#, mostrando o fatorial dos números,com um arquivo da copilação do projeto
Já na segunda pasta (testeBRF1) está localizada os arquivos dos testes unitarios com a referia ao projeto
da pasta BRF1, que está localizada o codigo do desafio.
